# ImageStudio Alignment Project

This project automates aligning blue-outline circles to black/light-grey data dots in ImageStudio.

## Folder Structure

```
~/opencode-imagestudio/
├── screenshots/       <- drop user screenshots here
├── setup.py           <- STEP 1: detect dots + blues, save pairs
├── align.py           <- STEP 2: drag each blue to its data dot
├── targets.json       <- generated by setup.py, consumed by align.py
├── archive/           <- old failed scripts (ignored)
└── README.md          <- this file
```

## Workflow (Two Steps)

### Step 0: Prepare
1. In ImageStudio, open file `0000300_01` (or whatever the target image is)
2. Make sure the full grid is visible (zoom/scroll as needed)

### Step 1: Detect
1. **Hide the blue-outline circles** in ImageStudio (toggle Grid Array visibility)
2. **Take a screenshot** of the ImageStudio window:
   - On macOS: `Cmd+Shift+4`, then drag to select the ImageStudio window
   - **Save to `~/opencode-imagestudio/screenshots/dots.png`**
3. **Show the blue-outline circles again** (toggle back on)
4. From a terminal: `cd ~/opencode-imagestudio && python3 grid_detect.py`
5. The script will:
   - Detect high-confidence Hough anchors
   - Learn the uneven row structure and local column structure
   - Save all 240 definitive modeled positions to `predicted_positions.json`
6. With the blue outlines visible, run: `python3 prepare_targets.py`
7. The target-preparation script will:
   - Capture the current ImageStudio window
   - Detect the visible blue outlines
   - Pair each blue with a unique position from the complete 240-position model
   - Save the pairs to `targets.json`

### Step 2: Align
1. From the same terminal: `python3 align.py`
2. The script will:
   - Verify the window hasn't moved
   - For each pair: double-click to select the blue, drag to the data dot, click to deselect
   - For extras: drag them to a corner (so they don't interfere)
   - Use **long delays** (~0.7s between actions) to avoid lag
   - Only click **inside the blue rectangle** (edge safety)

## Recovery

If the alignment goes wrong:
- **Cmd+Z** in ImageStudio to undo recent moves
- If the rectangle got shrunk, **Cmd+Z** multiple times to restore it
- For a clean reset: close the file in ImageStudio, then reopen

## Files Generated During Run

- `/tmp/dots.png` — your screenshot (read by setup.py)
- `/tmp/blues.png` — current view (written by setup.py)
- `targets.json` — pairs + extras (written by setup.py, read by align.py)
